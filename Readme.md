# map-glyph-server

Server for glyph/font files compatible with mapbox-gl-js

## Install

```sh
$ npm install --save map-glyph-server
```

## Usage

```js
var mapGlyphServer = require('map-glyph-server');

mapGlyphServer('Rainbow');
```
